bob16

A simple Pygame-like JS game engine.
Built using the JS Canvas API. It has its own editor at [The bob16 editor](https://maksydab.github.io/bob16/). It includes features such as modifying code in Lua and creating art, then using that art to create tilemaps. The unique feature is that it saves all game data into PNG files.

Ubabuba

It’s my first fully finished game (I'm not really proud of it though; it was put together in less than a week or two during free time). [Ubabuba on itch](https://maksydab.itch.io/ubabuba)
